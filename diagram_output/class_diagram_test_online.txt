Untuk testing Class Diagram yang telah diperbaiki, Anda dapat:

1. **PlantUML Online Editor:**
   - Buka: http://www.plantuml.com/plantuml/uml/
   - Copy-paste isi file: diagram_uml/class_diagram_pos_system.puml
   - Klik "Submit" untuk melihat hasil render

2. **PlantUML Web Server:**
   - Buka: https://plantuml-editor.kkeisuke.com/
   - Paste kode PlantUML
   - Real-time preview

3. **VS Code Extension:**
   - Install "PlantUML" extension
   - Buka file .puml
   - Tekan Alt+D untuk preview

4. **Draw.io Integration:**
   - Buka draw.io
   - File → Import → Pilih "PlantUML"
   - Paste kode untuk konversi
